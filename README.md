## 📖 项目概述
本项目为Python语言与数据分析课程完整大作业，搭建**群体特征-职业生态-城市适配**三维量化分析体系，基于全球多权威公开数据源，完成数字游民全流程数据分析：模拟多渠道数据爬取、数据清洗归一化、特征工程、多维度可视化绘图，输出完整研究报告与可视化图表，量化揭示数字游民人口画像、收入分层、城市选择核心影响因子。

## ✨ 项目四大核心模块
1. **多源数据爬取模块**
模拟MBO Partners、Localyze、国内文旅平台、海外媒体多渠道数据采集，统一结构化原始数据集。
2. **数据预处理模块**
单位统一、异常值剔除、加权均值补全缺失值、收入/年龄分层特征构建、0-100无量纲标准化。
3. 多维统计分析模块
分三大维度：人群画像、职业收入分层、城市四维适配度量化评分。
4. **专业可视化模块**
自动生成5类分析图表：核心挑战柱状图、收入满意度散点、城市适配雷达图、职业收入热力图、城市成本关联散点图，高清300DPI导出。

## 🧰 运行环境与依赖
### 1. 前置Python版本
Python 3.8 及以上
### 2. 一键安装依赖
pip install numpy pandas matplotlib
### 3. 本地运行命令
python 数字游民.py
#### 运行后自动输出 csv 结果文件 + 5 张高清 PNG 图表至本地文件夹。

## 📊 数据分析核心维度
### 1.群体特征维度：年龄分层、性别占比、学历分布、消费与生活模式
### 2.职业收入维度：行业分布、收入分层、AI 工具使用与收入相关性
### 3.城市适配维度：政策支持 / 生活成本 / 基建 / 文化包容四维加权评分，对比全球热门城市
